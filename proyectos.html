    Introducción

    Para que una máquina pueda funcionar necesita que alguien lo maneje, por ejemplo: Un tractor o un automóvil necesita de un operador que mueva correctamente los controles para un buen funcionamiento. Otras máquinas por ejemplo una calculadora necesitan que alguien les de una serie de instrucciones para que, al momento de que un usuario requiera realizar una operación aritmética la calculadora sepa cual operación se está solicitando e imprima el resultado correcto. Además, en el caso de las páginas internet, requieren tener un diseño y un sistema de interacción con los usuarios para que se sientan atraídos y puedan navegar en tu página web con mayor facilidad. Esto por ejemplo se puede hacer diciendole a la computadora que debe imprimir en la pantalla por medio de instrucciones las cuales, al momento de ejecutarse la pantalla del ordenador o del celular, impriman ciertas figuras, colores y animaciones que hacen atractiva la página asi como de hipervinculos parala navegación de una página a otra. Todo lo anterior se logra con la programación.

    La PROGRAMACIÓN es el diseño e implementación de una estructura que consta de una serie de instrucciónes paso por paso para una realizar una acción y/o acciones determinada. Estas instrucciónes son conocidas como algoritmos.

    El diseño e implementación de estos algoritmos son vitales para la programación de computadoras pues estas carecen de la capacidad de razonamiento y no son capaces de entender instrucciones implicitamente.

    Por ejemplo si te dan la instrucción "Haz un omelete" tu con tu experiencia sabes que un omelette esta hecho de huevo por lo tanto solo necesitas prender la estufa, tomar un sartén y ponerlo en la flama, agregar aceite al sartén, sacar dos o más huevos del refrigerador batirlos en un recipiente y echar el batido en el sartén esperar un momento a que se cosa un lado para voltear tu omelette con una espatula para que el otro lado se cosa. Finalmente ya que ambos lados estan perfectamente cocidos colocas tu omelette en un plato y esta listo para servir.

    Todo el párrafo anterior si tu se lo das a leer a tu familia sabe y entiende perfectamente que es una receta para un omelette, a menos que nunca haya echo un omelette en su vida. Ahora si le das la misma instrucción a una máquina "Haz un omelette", la máquina no entenderá ni siquiera la primera palabra, porque no tiene la información necesaria, no sabe lo que es un omelette ni lo que es un huevo o lo que es un sartén por lo tanto requiere que se le describa prefectamente todo lo que necesita así como para que sirve cada cosa. Ahora una vez dada las características de lo que va a utilizar necesita un algoritmo es decir una serie de pasos bien estructurados para que pueda hacer un omelette.

    Una vez que ya le diste las cacracterísticas de los objetos que va a utilizar y el algoritmo que diseñaste para esa tarea ahora si, cuando le digas a la máquina "Haz un omelette" sabrá que objetos se utilizan y como usar paso por paso cada uno de ellos para lograr su objetivo.

    En conclusión, a una máquina hay que describirle una serie de "objetos" dandoles atributos y características para que sean facilmente reconocibles y una vez descritos estos "objetos" necesitas diseñar un algoritmo, una serie de pasos para que la máquina haga lo que tu desees. Este tipo de programación se conoce como "Programación orientada a objetos" donde un objeto puede ser cualquier cosa a la que quieras dotarle ciertas características y acciones. La POO es utilizada en dos lenguajes importantes a nivel mundial como lo son Python y Java con los cuales vamos a experimentar más adelante.

    Actualmente la tecnología ha avanzado tanto que hemos mejorado nuestro diseño de algoritmos y hemos dotado a las máquinas de herramientas para que hagan varias cosas en automático, por ejemplo manejar un automóvil. Es importante potenciar nuestros conocimientos en programación y mejorar nuestros algoritmos así como mejorar nuestra ceratividad para diseñar páginas web para lograr una eficiente implementación de las máquinas que se construyan en el futuro para mejorar nuestra comunicación y bienestar y el de los demás.

    Python

    Python es un lenguaje de programación interpretado, funcional, orientado a objetos e interactivo. Comunmente es comparado (dando la preferencia) a Lisp, Tcl, Ruby, Perl, C#, Visual Basic, Visual Fox Pro, Scheme, Java o MatLab... y es mucho más famoso, potente y multifuncional.     HTML y CSS

    HTML
    HTML, que significa Lenguaje de Marcado para Hipertextos (HyperText Markup Language) es el elemento de construcción más básico de una página web y se usa para crear y representar visualmente una página web. Determina el contenido de la página web, pero no su funcionalidad. Otras tecnologías distintas de HTML son usadas generalmente para describir la apariencia/presentación de una página web (CSS) o su funcionalidad (JavaScript). HTML le da "valor añadido" a un texto estándar en español. Hiper Texto se refiere a enlaces que conectan una página Web con otra, ya sea dentro de una página web o entre diferentes sitios web. los vínculos son un aspecto fundamental de la Web. Al subir contenido a Internet y vincularlo a páginas de otras personas, te haces participante activo de esta Red Mundial.***

    CSS

CSS describe como debe ser renderizado el elemento estructurado en pantalla, en papel, hablado o en otros medios. ****
